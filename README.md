# minecraft-server-status

## 简介 

我的世界状态显示

包括

- 服务器信息
  - 服务器 Logo
  - 服务器名称
  - 服务器介绍
  - 服务器版本
  - 服务器是原版还是 Mod
- 玩家信息
  - 在线人数
  - 玩家名称
  - 玩家皮肤
  - 玩家头像

## 如何使用

1. 克隆
2. 我的世界服务端里的 server.properties，enable-query 修改为 true
3. 修改 index.php 里面的服务器 IP、端口、查询端口
4. 访问部署后的网站即可
